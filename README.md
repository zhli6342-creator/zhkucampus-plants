# 药用植物学 · 校园实习小助手

> 一款专为药用植物学校园实习教学设计的移动端网页应用，收录校园植物 117 种，支持检索、筛选、收藏与详情查看。

---

## 功能介绍

- 🌿 **植物图鉴浏览** — 卡片式展示全部 118 种校园植物
- 🔍 **多维度检索** — 支持按物种名、科属、拉丁名、辨识特征、药用功效关键词搜索
- 🌳 **生活型筛选** — 乔木、灌木、草本、藤本、蕨类、棕榈、裸子植物一键筛选
- 💊 **功效分类** — 清热解毒、活血化瘀、祛风除湿、止血收敛等八大类快速定位
- 📖 **物种详情** — 含辨识特征、中药药用记载、多张图片轮播
- 🔖 **收藏功能** — 本地保存收藏物种，方便复习
- ⚠️ **有毒标注** — 含毒成分植物红框高亮警示

---

## 软件架构

纯前端静态网页，无需服务器，可直接部署于 Gitee Pages / GitHub Pages。

```
项目根目录/
├── index.html          # 主页面（含全部样式与交互逻辑）
├── plants.js           # 植物数据文件（117种）
└── images/             # 植物图片目录
    ├── 001_沿阶草.jpg
    ├── 002_栀子.jpg
    └── ...             # 命名格式：编号（三位）_中文名.jpg
```

技术栈：HTML5 + CSS3 + 原生 JavaScript，无任何外部框架依赖。

---

## 图片更新说明

在 `images/` 目录中按以下规则存放图片即可自动关联：

| 规则 | 示例 |
|------|------|
| 命名格式 | `编号_中文名.jpg` |
| 编号补零 | `001_沿阶草.jpg`、`010_海南红豆.jpg` |
| 多图支持 | `002_栀子_2.jpg`、`002_栀子_3.jpg` |
| 推荐尺寸 | 宽 600–1200px，单张 200KB 以内 |

多图需在 `plants.js` 中对应物种的 `images` 数组中补充路径。
